A 100% free, limitless site migration and backup plugin. Export, import, and do direct site-to-site transfers without file size limits.

== Description ==

Open Migration is a modern, free, and unlimited solution to backup or migrate your WordPress site. 

Unlike premium plugins that arbitrarily limit your upload sizes to force you into paid upgrades, Open Migration uses intelligent chunking. This bypasses server limits, allowing you to upload 5GB+ archives on even the cheapest shared hosting.

= Features =
* Unlimited File Sizes
* Direct Site-to-Site Transfer via REST API
* Modern, beautiful UI

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/open-migration` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Open Migration menu to export or import your site.

== Frequently Asked Questions ==

= Is this really unlimited? =
Yes. We bypass upload limits by slicing the upload into chunks within the browser.
